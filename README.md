# Hybrid Recommender Core

This repository contains the main backend and algorithm work for the recommender system:

- hybrid recommendation scoring
- genetic algorithm optimization
- dataset cleaning pipeline
- offline evaluation logic
- Flask integration entry points

## Main Files

- `src/recommender.py`
- `src/data_cleaning.py`
- `src/evaluation.py`
- `app.py`
- `wsgi.py`

## Dependencies

Install with:

```bash
pip install -r requirements.txt
```
